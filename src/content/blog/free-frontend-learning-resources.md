## 七、免费学习路线图

如果你不知道从哪里开始学，跟着这个路线走：

### 🟢 阶段一：HTML + CSS（2-4 周）

```
HTML 基础标签 → HTML5 语义化 → CSS 基础 → Flexbox → Grid
→ 响应式设计 → CSS 动画 → 做一个静态页面
```

**资源**：MDN + B站"HTML+CSS 入门"

### 🟡 阶段二：JavaScript 基础（4-6 周）

```
变量/数据类型 → 函数 → 数组/对象 → DOM 操作 → 事件处理
→ ES6+ 语法 → 异步编程 → 做一个交互页面
```

**资源**：JavaScript.info + B站"JavaScript 高级"

### 🟠 阶段三：前端框架（6-8 周）

```
选 Vue3 或 React（建议先学一个）→ 组件化开发 → 状态管理
→ 路由 → HTTP 请求 → 做一个完整项目
```

**资源**：官方文档 + B站 对应教程

### 🔴 阶段四：工程化与进阶（4-8 周）

```
Git → npm → Webpack/Vite → TypeScript → 单元测试
→ 性能优化 → 做 2-3 个项目
```

**资源**：各工具官方文档 + 掘金文章

### 🟣 阶段五：求职准备（4-8 周）

```
刷算法题（LeetCode 热题 100）→ 复习基础知识 → 准备项目介绍
→ 模拟面试 → 投简历
```

**资源**：代码随想录 + 牛客网面经

## 八、我的学习建议

### 8.1 免费资源的正确用法

1. **不要贪多**：选 1-2 个资源，学完再换，不要同时看 5 个教程
2. **一定要动手**：看教程≠学会了，手敲代码才算
3. **做笔记**：用 Notion/Obsidian 记录学习笔记，方便复习
4. **做项目**：每学完一个知识点，就做一个相关的小项目
5. **教别人**：写博客、做分享，教是最好的学

### 8.2 什么情况下可以考虑付费

免费资源虽然好，但以下情况付费课程可能更高效：

- **需要系统学习路径**：有人帮你规划学习路线
- **需要答疑**：遇到问题有人解答
- **需要项目实战**：跟着做商业级项目
- **需要面试指导**：模拟面试+简历修改

但前提是：**你已经充分使用了免费资源，并且确认自己真的学不下去**。
